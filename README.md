##PhantomDB Assistant

** (tela de login e senha )

Este projeto simula um sistema de login utilizando apenas front-end ou seja login e senha são FICTÍCIOS e não reais, não expondo nenhum dados sensível!! 

⚠️ Não possui backend — autenticação apenas demonstrativa.

Ideia do Projeto


Desenvolvi o PhantomDB Assistant, um projeto pessoal focado em simular como empresas podem usar IA para interagir com bancos de dados de forma simples e inteligente.

A proposta é transformar dados (mesmo que fictícios) em conversas úteis, permitindo consultar, editar e gerenciar informações através de um chatbot, sem precisar de conhecimento técnico em SQL.

Na prática, a solução mostra como é possível:

Automatizar consultas a banco de dados
Centralizar gestão de informações
Facilitar tomada de decisão com respostas rápidas

💡 Impacto e resultados esperados:
Mais agilidade operacional, redução de erros manuais e democratização do acesso a dados dentro de equipes.
